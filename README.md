# V.GRAPHIKS | Creative Designs

Site institucional da V.GRAPHIKS, criado para apresentar servicos criativos, projetos recentes e facilitar o contacto com potenciais clientes.

## Visao Geral

Este site e uma landing page responsiva para um estudio criativo focado em design, desenvolvimento web e experiencias digitais.

Permite:
- Apresentar a marca V.GRAPHIKS
- Mostrar os principais servicos do estudio
- Exibir projetos recentes
- Apresentar informacao sobre a empresa
- Disponibilizar um formulario de contacto
- Alternar o idioma entre ingles e portugues

## Tecnologias

- **HTML5**: estrutura das secoes do site
- **CSS3**: estilos, layout responsivo, gradientes e animacoes
- **JavaScript**: menu mobile, troca de idioma, animacoes ao scroll, formulario e notificacoes
- **Google Fonts**: utilizacao da fonte Poppins

## Como Abrir

Como o projeto e um site estatico, nao e necessario instalar dependencias.

Para visualizar:

1. Abrir a pasta do projeto
2. Abrir o ficheiro `index.html` no navegador

Tambem pode ser usado o Live Server do Visual Studio Code para uma experiencia mais pratica durante o desenvolvimento.
