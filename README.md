# V.GRAPHIKS | Creative Designs

Site institucional da V.GRAPHIKS, criado para apresentar serviços criativos, projetos recentes e facilitar o contacto com potenciais clientes.

## Visão Geral

Este site é uma landing page responsiva para um estúdio criativo focado em design, desenvolvimento web e experiências digitais.

Permite:
- Apresentar a marca V.GRAPHIKS
- Mostrar os principais serviços do estúdio
- Exibir projetos recentes
- Apresentar informações sobre a empresa
- Disponibilizar um formulário de contacto
- Alternar o idioma entre inglês e português

## Tecnologias

- **HTML5**: estrutura das secções do site
- **CSS3**: estilos, layout responsivo, gradientes e animações
- **JavaScript**: menu mobile, troca de idioma, animações no scroll, formulário e notificações

## Como Abrir

Como o projeto é um site estático, não é necessário instalar dependências.

Para visualizar:

1. Abrir diretamente através do link `nashi34.github.io/V.GRAPHIKS/`
2. Abrir a pasta do projeto
3. Abrir o ficheiro `index.html` no navegador

Tambem pode ser usado o Live Server do Visual Studio Code para uma experiencia mais pratica durante o desenvolvimento.
